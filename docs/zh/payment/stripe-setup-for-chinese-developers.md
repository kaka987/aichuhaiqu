# 中国开发者接入 Stripe：从注册到第一笔入账

Stripe 是海外独立开发者收款的事实标准。接口文档清晰、支持订阅管理、全球覆盖 40+ 个国家的支付方式。但对中国大陆开发者来说，第一个问题就是：**Stripe 不支持中国大陆主体注册。**

不支持的意思是：你不能用中国大陆的个人身份或中国大陆公司去注册 Stripe 账号。必须有一个 Stripe 支持地区的公司实体。

这篇讲清楚几条可行路径，以及每条路的实际体验。

## 路径一：美国 LLC + Stripe US

这是目前中国独立开发者用得最多的方案。

**前置条件：**
- 一家美国 LLC（推荐 Wyoming）
- EIN（雇主识别号）
- 美国银行账户（Mercury / Relay）

**注册流程：**
1. 打开 stripe.com，点击 "Start now"
2. 选择国家：United States
3. 填写公司信息：LLC 名称、EIN、注册地址（用 Registered Agent 地址）
4. 填写个人信息：你的姓名、护照号、居住地址（填中国的真实地址）
5. 绑定银行账户：输入 Mercury/Relay 的 routing number 和 account number
6. 填写业务信息：产品描述、网站 URL、预计月交易额
7. 提交审核

**审核注意事项：**
- 网站必须已经上线，至少有产品介绍页和定价页
- 业务描述要具体。"SaaS" 不够，"AI-powered email writing tool, monthly subscription $9-29/month" 才够
- Stripe 可能会额外要求上传 LLC Certificate 或 EIN 确认信的照片
- 审核通常 1-2 个工作日，偶尔 3-5 天

**费率（截至 2026 年 6 月）：**

| 项目 | 费率 |
|------|------|
| 美国卡在线支付 | 2.9% + $0.30/笔 |
| 国际卡 | +1.5% |
| 货币转换 | +1% |
| Stripe Billing（订阅管理） | 0.5%（基于 recurring 交易额） |
| 到账周期 | T+2（美国银行账户） |

一笔 $29 的月订阅，如果用户是美国人用美国卡，实际扣费大约 $1.14（2.9% + $0.30），你到手 $27.86。如果用户是欧洲人用欧元卡，扣费大约 $1.87（2.9% + 1.5% + 1% + $0.30），到手 $27.13。

## 路径二：香港公司 + Stripe HK

如果你已经有香港公司，可以直接注册 Stripe Hong Kong。

**前置条件：**
- 香港公司（CR 注册证 + BR 商业登记证）
- 香港银行账户（汇丰、渣打、ZA Bank 等）

**和 Stripe US 的区别：**

| 对比项 | Stripe US | Stripe HK |
|--------|-----------|-----------|
| 公司要求 | 美国 LLC/Corp | 香港有限公司 |
| 银行要求 | 美国银行 | 香港银行 |
| 基础费率 | 2.9% + $0.30 | 3.4% + HK$2.35 |
| 结算货币 | USD | HKD |
| 到账周期 | T+2 | T+7（新账号），后降至 T+2 |

Stripe HK 的费率比 US 贵一点，到账周期新账号也更长。但好处是港币结算后转回国内比较方便（香港银行 → 内地银行转账通道成熟）。

**香港银行开户的坑：** 汇丰、渣打这类传统银行现在对小型公司开户越来越严，可能需要面签或者提供详细的商业计划。ZA Bank（众安银行）和 Airstar Bank（天星银行）是纯线上开户的虚拟银行，门槛低很多，但部分功能有限制。

## 路径三：Stripe Atlas 一站式

前面提过，Stripe Atlas 花 $500 帮你搞定 LLC + EIN + Stripe + Mercury 全套。如果你从零开始，这是摩擦最小的路径。

**Stripe Atlas 包含：**
- Wyoming 或 Delaware LLC 注册
- EIN 申请
- Stripe 账号（自动审核通过，因为是 Atlas 内部流程）
- Mercury 银行账号开户邀请
- Operating Agreement 模板
- 一年 Registered Agent 服务
- 税务指导文档（不是报税服务）

**Atlas 的缺点：**
- 贵。自己走流程 $200-225，Atlas 要 $500
- LLC 注册在 Delaware（默认），年审费 $300 比 Wyoming 的 $60 贵
- 续年的注册代理服务费另算

但如果你算时间成本（自己折腾 2-4 周 vs Atlas 帮你搞 1-2 周），而且 Stripe 审核几乎零风险（Atlas 注册的公司 Stripe 不会拒），$500 不算贵。

## 钱到了 Stripe，怎么回国内

Stripe 把钱打到你的 Mercury 银行账户后，你需要把美金转回国内用。常见路径：

**路径 A：Mercury → Wise → 国内银行卡**

1. Mercury 账户发起 Wire Transfer 到你的 Wise USD 账户
2. Wise 换汇（USD → CNY），汇率接近中间价，手续费约 0.4%-0.6%
3. Wise 转账到你的国内银行卡（支持支付宝、银行卡）
4. 到账时间：1-2 个工作日

**路径 B：Mercury → Payoneer → 国内银行卡**

1. Mercury Wire 到 Payoneer 美元账户
2. Payoneer 提现到国内银行卡，费率约 1.2%
3. 到账时间：1-2 个工作日

**路径 C：香港银行 → 内地银行**

如果你走 Stripe HK 路径，港币到香港银行后：
1. 通过 FPS / 汇丰内地账户 / 中银快汇 转入内地
2. 费率接近零（汇丰跨境账户之间免手续费）
3. 年度结汇限额：等值 5 万美元（个人）

**费率对比：**

| 路径 | 总费率（含汇损） | 到账时间 | 适用场景 |
|------|-----------------|---------|---------|
| Wise | 0.4%-0.6% | 1-2 天 | 最便宜，适合大额 |
| Payoneer | 1.2% | 1-2 天 | 操作简单，适合懒人 |
| 香港银行 | ≈0% | 即时-1天 | 有香港账户的人 |

**注意事项：**
- 个人年度结汇限额 5 万美元。超过的部分需要提供贸易背景证明（合同、Invoice 等）
- Wise 每笔转账有限额（具体因账户等级不同），大额分多笔转
- 所有收入都要依法报税。具体怎么报、报多少，建议找一个懂跨境税务的会计

## 常见问题

**Q：Stripe 注册时居住地址填中国会不会有问题？**

不会。Stripe 允许公司在美国注册但 beneficial owner（实际控制人）住在其他国家。填你在中国的真实居住地址就行。

**Q：Stripe 会因为我是中国人而拒绝吗？**

不会，只要你的公司实体在 Stripe 支持的国家。Stripe 关心的是公司注册地，不是你的国籍。但如果你的业务涉及 Stripe 限制类目（成人内容、加密货币交易、赌博等），会被拒。

**Q：没有产品也能先注册 Stripe 吗？**

技术上可以，但审核可能会要求你提供网站。建议至少有一个能打开的 Landing Page，上面有产品介绍和定价。一个简单的单页就够了。

**Q：Stripe 账号会被封吗？**

会。常见原因：退款率过高（>1%）、违禁业务、提供虚假信息、交易模式异常（突然从每月 $100 跳到 $10000）。正常做业务、退款率控制好，不会有问题。

**Q：PayPal 不行吗？**

PayPal 支持中国大陆个人和企业注册（PayPal 中国），但 PayPal 的问题是：费率高（4.4% + 固定费 for 国际交易）、冻结资金的概率大、提现到国内麻烦。对于 SaaS 和数字产品来说，Stripe 的体验远好于 PayPal。PayPal 更适合作为备选支付方式（给不愿意用信用卡的用户一个选择），而不是主力收款通道。
