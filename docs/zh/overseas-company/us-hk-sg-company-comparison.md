# 美国、香港、新加坡：三地公司怎么选

出海要注册海外公司，最常见的选择就是美国、香港和新加坡。三个地方各有优势，没有绝对的「最优解」——取决于你的产品类型、目标市场、预算和长期规划。

这篇把三者的核心差异拉出来对比，帮你做决策。

## 总览对比表

| 维度 | 美国 LLC (Wyoming) | 香港有限公司 | 新加坡私人有限公司 |
|------|-------------------|-------------|-------------------|
| **注册费用** | $200-500 | ¥3,000-6,000 | S$800-2,000 |
| **注册时间** | 1-3 天（州批准） | 1-2 天（电子注册） | 1-2 天（电子注册） |
| **年维护成本** | $160-200/年 | ¥3,000-5,000/年 | S$1,500-3,000/年 |
| **企业所得税** | 0%（LLC 穿透不交联邦企业税） | 8.25%-16.5%（两级制） | 17%（首 20 万 SGD 有减免） |
| **银行开户难度** | 低（Mercury 全线上） | 中高（可能需面签） | 中（部分银行线上开户） |
| **Stripe 支持** | ✓ Stripe US | ✓ Stripe HK | ✓ Stripe SG |
| **适合阶段** | 早期独立开发者 | 有稳定收入后 | 想做东南亚市场或融资 |

## 美国 LLC：独立开发者的首选

### 为什么大家选美国

**税务简单。** 单人 LLC 在美国税务体系里是 "disregarded entity"——公司收入直接穿透到个人。如果你是非美国税务居民（人在中国），美国 LLC 的收入来源如果不来自美国境内的活动，通常不需要向美国缴联邦所得税。你只需要每年报一个信息申报表（Form 5472 + Pro Forma 1120）。

注意：这不是逃税，是合法的税务架构。但需要确保合规申报，建议找懂跨境税务的会计。

**成本最低。** Wyoming LLC 首年 $200-500，之后每年 $160-200 维护。对刚起步的独立开发者来说，这个成本可以忽略不计。

**银行开户容易。** Mercury 全线上开户，不需要面签，不需要飞美国。对比香港银行动辄要求面签，方便太多。

**支付平台全兼容。** Stripe US、PayPal US、Wise Business——所有主流平台都支持。

### 美国 LLC 的局限

- **如果你要融资：** 投资人通常要求 Delaware C-Corp，不是 LLC。LLC 转 C-Corp 需要额外操作和费用。
- **如果你主要做东南亚市场：** 美国公司在东南亚没有税务优势，新加坡可能更合适。
- **年报和税务申报：** 虽然可能不交税，但申报义务还在。不报的后果是罚款。

## 香港有限公司：中国人最熟悉的选择

### 为什么选香港

**离中国最近。** 同一时区、中文通行、法律体系熟悉。资金回国方便——香港银行和内地银行之间的转账通道成熟（汇丰跨境账户、中银快汇等）。

**税率低且简单。** 香港实行属地原则——只有来源于香港的利润才需要纳税。如果你的产品面向全球用户、服务器不在香港、运营团队不在香港，可以申请「利得税豁免」（Offshore Claim）。能批下来的话，有效税率为零。

利得税税率：
- 首 200 万港币利润：8.25%
- 超过部分：16.5%

**适合已有稳定收入的阶段。** 香港公司的维护成本比美国 LLC 高（年审 + 审计 + 报税），如果你还在验证产品阶段，用美国 LLC 更经济。

### 香港公司的坑

**银行开户越来越难。** 2020 年以后，香港银行对小型公司（尤其是没有实际办公地址的）开户审核明显收紧。汇丰、渣打这类大行可能要求面签、提供详细的商业计划书、证明资金来源。ZA Bank 等虚拟银行门槛低一些，但部分功能有限。

**审计是强制的。** 香港法律要求所有有限公司每年做审计（由持牌会计师出具审计报告）。即使你全年零收入也要做。审计费 ¥3,000-8,000/年不等，加上秘书公司费用，年维护成本比美国 LLC 高出不少。

**离岸豁免不一定批。** 申请利得税豁免需要提供大量证据证明你的业务确实不在香港运营。税务局审核严格，不是说申请就能批。如果没批下来，正常按 16.5% 交税。

### 费用明细

| 项目 | 费用 |
|------|------|
| 注册（含秘书公司服务） | ¥3,000-6,000 |
| 年审（Annual Return） | ¥1,500-2,500 |
| 审计（由 CPA 出具） | ¥3,000-8,000 |
| 秘书公司年费 | ¥1,000-3,000 |
| **年维护总计** | **¥5,500-13,500** |

## 新加坡私人有限公司：东南亚门户 + 融资友好

### 为什么选新加坡

**国际声誉好。** 新加坡在全球商业信用排名常年前三，银行开户、与国际合作伙伴签约时「新加坡公司」的信任度比「美国 LLC」或「香港公司」更高。

**融资友好。** 如果你有融资计划，新加坡私人有限公司（Pte Ltd）的法律结构被国际投资人广泛接受。很多出海公司（包括 Shein 的海外实体）注册在新加坡。

**东南亚市场入口。** 做东南亚市场（印尼、泰国、越南、菲律宾），新加坡公司是最方便的跳板——税收协定、银行通道、法律体系都有优势。

**税务优惠。** 新加坡对新公司有 Startup Tax Exemption：前三年里，每年首 10 万 SGD 利润的 75% 免税，接下来 10 万 SGD 的 50% 免税。实际有效税率可以低到 5% 左右。

### 新加坡公司的坑

**年维护成本最高。** 强制审计（年收入超过 S$10M 的公司，或不满足小型公司豁免条件的）、公司秘书、注册地址、年度报税——加起来费用比美国和香港都高。

不过对大多数独立开发者来说，如果满足 "小型公司" 标准（年收入 < S$10M、员工 < 50、资产 < S$10M，满足两项即可），可以豁免审计，只需要编制未经审计的财务报表。

**需要本地董事。** 新加坡法律要求至少一名本地居民董事（新加坡公民、永久居民或持有 Employment Pass 的人）。如果你人在中国，需要找一个代名董事（Nominee Director）——这是一项额外费用（S$1,500-3,000/年）。

**银行开户。** DBS、OCBC 这类大行对远程开户的非居民公司审核较严。但 Aspire、Airwallex 等金融科技平台提供线上开户，门槛低很多。

### 费用明细

| 项目 | 费用 |
|------|------|
| 注册（含代理费） | S$800-2,000 |
| 注册地址 | S$300-800/年 |
| 公司秘书 | S$300-600/年 |
| 代名董事（如需要） | S$1,500-3,000/年 |
| 年度报税（IRAS） | S$300-800 |
| 审计（如不豁免） | S$1,500-3,000 |
| **年维护总计（有代名董事）** | **S$2,400-5,200** |
| **年维护总计（无代名董事）** | **S$900-2,200** |

## 怎么选：决策树

```
你现在的情况是？
│
├── 刚起步，月收入 < $1,000
│   └── 美国 Wyoming LLC（成本最低，够用）
│
├── 有稳定收入，主要面向欧美用户
│   ├── 不打算融资 → 美国 LLC（税务简单，维护便宜）
│   └── 有融资计划 → 新加坡 Pte Ltd 或美国 Delaware C-Corp
│
├── 主要面向东南亚市场
│   └── 新加坡 Pte Ltd
│
├── 需要资金频繁回国
│   └── 香港公司（回款最方便）
│
└── 团队在中国，想要一个「看起来正式」的海外实体
    └── 新加坡（国际信用最好）或香港（最方便）
```

## 能不能注册多个

可以，而且很多做到一定规模的出海公司确实有多个实体：

- **美国 LLC：** 用来挂 Stripe US，收北美市场的钱
- **香港公司：** 用来做亚太区的资金中转，方便回国
- **新加坡公司：** 用来做对外的品牌主体，签合同

但这是有规模之后的事。刚起步的独立开发者，**一个美国 LLC 足够。** 等年收入过了 $50,000-100,000 再考虑多实体架构。多一个公司就多一套年审、报税、银行维护——管理成本不能忽略。

## 常见误区

**「香港公司可以零申报省钱」**

技术上可以做零申报，但前提是你的公司确实没有任何业务活动。如果有收入但做了零申报，被税务局查到就是偷税——罚款、补税、甚至追究刑事责任。不要为了省审计费冒这个险。

**「新加坡税率 17% 太高了」**

17% 是名义最高税率。新创企业前三年有大幅减免，实际有效税率可以低到 5%。而且如果你的利润在海外（不汇入新加坡），部分情况下可以适用免税。具体需要咨询税务顾问。

**「美国 LLC 不需要报税」**

不对。LLC 虽然可能不需要**交**税（非美国来源收入对非居民免联邦所得税），但**申报**义务是有的。每年要提交 Form 5472 + Pro Forma 1120。不报会被罚款——每年 $25,000 起。

**「注册了公司就能开所有银行」**

不是。每个银行有自己的风控标准。香港的汇丰可能拒绝你，新加坡的 DBS 也可能拒绝你。被拒不代表你的公司有问题，只是没满足那家银行的偏好。多试几家，或者用 Mercury、Airwallex 这类对小公司友好的平台。
