# 跨境电商

- Shopify 独立站和亚马逊开店哪个更适合新手？
- 选品怎么做？有哪些数据工具？
- 独立站流量从哪来？
- 海外仓和一件代发（Dropshipping）怎么选？
- TikTok Shop 值得做吗？
