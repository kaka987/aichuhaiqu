# 有钱投流没钱做号：出海流量获取的两条路

出海产品的流量来源，归根结底就两种资源的投入：**钱**或者**时间**。

有预算，就投广告——Google Ads、Meta Ads、TikTok Ads。花钱买精准流量，当天就能看到效果。停止投放，流量立刻归零。

没预算（或者想省钱），就做内容——Twitter/X、YouTube、博客、Newsletter、Reddit。花时间做自媒体、写文章、参与社区。见效慢，但积累下来的流量是你的资产。

大多数独立开发者的现实情况是：**早期没什么钱，只能花时间；等产品赚到钱了，再用利润去投流加速。** 这篇把两条路都讲清楚，你根据自己的阶段选。

---

## 先想清楚一件事：你的产品适合哪种获客方式

不是所有产品都适合投广告，也不是所有产品都适合做内容。

| 产品类型 | 适合投流 | 适合做内容 |
|---------|---------|-----------|
| 高客单价 SaaS（$50+/月） | ✓ CPA 可承受 | ✓ 同时做 SEO |
| 低客单价工具（$5-10/月） | ✗ CPA 太高回不了本 | ✓ 靠自然流量降本 |
| 一次性付费产品（$20-50） | △ 看 LTV | ✓ SEO + 社区 |
| 免费工具 + 广告变现 | ✗ 不划算 | ✓ 必须靠自然流量 |
| B2B 软件 | ✓ LinkedIn Ads | ✓ 内容营销 |

**核心判断标准：你的 LTV（用户终身价值）能不能覆盖 CAC（获客成本）？**

如果你的产品月费 $29，用户平均用 8 个月，LTV 就是 $232。只要 CAC 低于 $232 的 1/3（约 $77），投流就是划算的。

如果你的产品月费 $5，用户平均用 3 个月，LTV 就是 $15。Google Ads 一个点击可能就要 $2-5，转化率 5% 的话 CAC 是 $40-100——远超 LTV。这种情况只能靠自然流量。

## 付费路线：用钱换效率

### Google Ads

**适合：** 用户会主动搜索你的品类的产品。搜索广告捕获的是「已有需求」的用户，转化率高。

**基本逻辑：**
- 你出价竞标关键词（比如 "AI writing tool"）
- 用户搜这个词时，你的广告出现在搜索结果顶部
- 按点击付费（CPC），不点不花钱

**独立开发者的起步建议：**

| 项目 | 建议 |
|------|------|
| 起步预算 | $10-30/天（测试阶段） |
| 关键词选择 | 从长尾词开始，避开大词 |
| 匹配方式 | 用 Phrase Match 或 Exact Match，不要用 Broad Match（烧钱太快） |
| 落地页 | 专门做一个和广告词强相关的页面，不要直接导首页 |
| 转化追踪 | 必须设置——注册、付费都要能追踪到来自哪个词 |

**常见错误：**
- 一上来就投大词（"AI tool"），一天烧几百美元没转化
- 不设转化追踪，不知道哪个词带来了付费用户
- 落地页和广告词不匹配（广告说 "email writer"，落地页是通用首页）

### Meta Ads（Facebook + Instagram）

**适合：** 产品有视觉化的使用场景，用户可能不会主动搜索但看到会感兴趣。

**和 Google Ads 的区别：** Google 是「人找广告」（用户搜了才看到），Meta 是「广告找人」（基于兴趣标签推送）。Google 转化率高但量有限，Meta 量大但需要好的创意素材来吸引注意力。

**起步建议：**
- 预算：$20-50/天
- 素材：视频效果远好于图片（3-15 秒竖版短视频）
- 定位：用兴趣标签 + Lookalike Audience
- 优化目标：前期跑 "Landing Page View"，有数据后切 "Purchase" 或 "Sign Up"

**2026 年的现状：** Meta Ads 的 CPM（千次展示成本）在持续上涨，尤其是美国市场。如果你的目标用户在东南亚、南美，CPM 会便宜很多。

### TikTok Ads

**适合：** 面向年轻用户（18-35 岁）的消费类产品、App、创意工具。

TikTok Ads 的起投门槛比 Google 和 Meta 低，2026 年最低日预算在 $20 左右。但它的转化链路比较长——用户在刷短视频的时候注意力不集中，从看到广告到完成注册的路径损耗比搜索广告大。

**适合做品牌曝光和 App 下载，不太适合做 SaaS 订阅的直接转化。**

### 付费路线的核心原则

1. **小预算测试，找到 winning creative 后再放量。** 不要一上来就砸 $1000/天。先用 $10-30/天测 3-5 组不同的创意/关键词/受众，跑 3-7 天看数据，留下 ROI 正的，砍掉其余的。
2. **追踪到底。** 花了 $100 进来 50 个注册，这 50 个人后续有几个付费了？如果不追踪到付费环节，你不知道广告到底赚不赚钱。
3. **预留 3 个月学费。** 投流前三个月基本在交学费——测素材、测受众、优化落地页。真正跑通模型一般在第 2-3 个月。如果预算只够投两周，建议别投，不够 ramp up 的。

## 免费路线：用时间换流量

没钱怎么办？花时间。做自媒体、写内容、混社区。见效慢，但成本低，积累下来的流量是你的长期资产。

### Twitter/X

**为什么优先做 Twitter/X：**
- 开发者、创业者、投资人高度集中
- 内容分发靠算法，不需要你有很多粉丝也能获得曝光
- 一条好的帖子可以带来几千到几万的 impression
- 可以直接和潜在用户互动

**怎么起步：**
- 定一个发帖节奏（每天 1-2 条）
- 内容方向：产品开发过程、出海经验分享、行业观察、技术干货
- 用英文发（面向全球用户）
- 参与别人的讨论——高质量的回复能给你带来 follower
- 不要只发广告，80% 内容提供价值，20% 提到自己的产品

**从 0 到 1000 follower 的时间：** 正常节奏大约需要 2-4 个月。如果某条帖子爆了，可能一周就涨几百。

### YouTube

**适合：** 教程类、工具演示、对比评测这类需要视觉展示的内容。

YouTube 的好处是内容保质期极长——一条好的教程视频发布一年后还能持续带来流量和订阅。而且 YouTube 视频在 Google 搜索结果中有独立展示位，等于同时做了视频 SEO 和 Google SEO。

**起步门槛：** 需要录屏软件（OBS 免费）、麦克风（$50 的 USB 麦就够）、简单的剪辑（DaVinci Resolve 免费）。不需要出镜，录屏 + 画外音就行。

**变现门槛：** YouTube 广告分成需要 1000 订阅 + 4000 小时观看。早期把 YouTube 当获客渠道，不要指望广告收入。

### 博客 + SEO

见上一篇《独立开发者 SEO 入门》。写博客是最经典的免费获客方式。区别是 SEO 的流量来自搜索引擎，不依赖粉丝数。

### Newsletter

**逻辑：** 把访客变成订阅者，建立一个你完全控制的触达渠道。不依赖任何平台算法。

**工具：** Beehiiv、Buttondown、ConvertKit。个人推荐 Beehiiv——免费版支持 2500 订阅者，功能够用。

**怎么起步：**
- 在你的网站/博客/Landing Page 上加一个邮件订阅入口
- 内容频率：每周或每两周一封
- 内容：行业动态、产品更新、实用教程的混合
- 目标：把一次性访客转化为长期读者，后续有新功能/新产品时直接触达

### Reddit / Indie Hackers / Hacker News

社区参与是免费获客中性价比最高的方式之一。一条好的 Reddit 帖子能带来几千次网站访问；一条 HN 首页帖子能带来上万。

但社区有规则，**硬广告会被秒删、被封号。** 正确的方式是：提供有价值的内容，自然地提到你的产品。

详细的社区和渠道盘点，见下一篇：[海外流量渠道和社区盘点](overseas-traffic-channels-and-communities.md)

## 混合打法：小预算 + 内容的组合拳

对大多数独立开发者来说，最现实的方案不是纯投流或纯做内容，而是混合：

**阶段一（0-3 个月）：纯内容**
- 产品刚上线，没钱投广告
- 做 Twitter、写博客、混社区
- 目标：验证产品方向、获取前 100 个用户

**阶段二（3-6 个月）：小预算测试**
- 产品有了稳定收入（$500-1000/月）
- 拿出 10%-20% 的收入测 Google Ads
- 同时继续做内容
- 目标：找到 CAC < LTV/3 的投放渠道

**阶段三（6 个月+）：放大**
- 投流模型验证通过
- 加大预算，同时加大内容产出
- SEO 流量开始起量（前面写的文章开始排名）
- 三条渠道并行：付费广告 + SEO + 社交媒体

**关键指标：**

| 阶段 | 核心指标 | 健康值 |
|------|---------|--------|
| 验证期 | 注册转化率 | > 3%（Landing Page → Sign Up） |
| 测试期 | CAC | < LTV/3 |
| 放大期 | Payback Period | < 3 个月 |

## 一个判断清单

不确定现在该投流还是做内容？问自己：

- **产品已经有付费用户了吗？** 没有 → 先做内容验证
- **LTV 算过了吗？** 没有 → 先搞清楚再说投流的事
- **有 $500+/月 的闲钱吗？** 没有 → 做内容
- **知道目标用户会搜什么关键词吗？** 知道 → Google Ads 适合你
- **产品有视觉化的使用场景吗？** 有 → Meta/TikTok Ads 适合你
- **你能坚持每周发 2-3 条有质量的内容吗？** 能 → 做 Twitter + 博客

两条路不矛盾。长期来看，最健康的状态是：有稳定的自然流量打底（SEO + 社交媒体），有付费广告做增量杠杆。
